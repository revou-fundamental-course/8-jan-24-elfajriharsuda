PERSEGI

Rumus Luas yaitu:

L = S x S

Dimana :

L = Luas

S = Sisi

Rumus Keliling yaitu:

K = 4 x S

Dimana :

K = Keliling

S = Sisi

Hitung Persegi


PERSEGI PANJANG

Rumus Luas yaitu:

L = p x l

Dimana :

L = Luas

p = Sisi Panjang

l = Sisi Lebar

Rumus Keliling yaitu:

K = 2 x p + 2 x l

Dimana :

K = Keliling

p = Sisi Panjang

l = Sisi Lebar

Hitung Persegi Panjang
