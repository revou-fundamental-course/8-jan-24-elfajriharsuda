PERSEGI

Rumus Luas yaitu:

L = S x S

Dimana :

L = Luas

Rumus Keliling yaitu:

K = 4 x S

Dimana :

K = Keliling

PERSEGI PANJANG

Rumus Luas yaitu:

L = p x l

Dimana :

L = Luas

p = Sisi Panjang

l = Sisi Lebar

Rumus Keliling yaitu:

K = 2p + 2l

Dimana :

K = Keliling

p = Sisi Panjang

l = Sisi Lebar
